## О проекте

SpeedTasks — это сборка небольших веб-приложений и компонентов, созданных для отработки навыков front-end разработки. Каждый проект представляет собой самостоятельный пример, демонстрирующий конкретные техники и подходы в веб-разработке.

## Список задач

| Файл | Описание |
|------|----------|
| `task1.html` | Анимированный переключатель (toggle switch) на чистом CSS |
| `task2.html` | Вращающийся 3D куб с цветными гранями |
| `task2,3.html` | 3D куб с управлением цветом через RGB слайдеры |
| `task3.html` | Генератор RGB цветов с предварительным просмотром |
| `task4.html` | Применение CSS фильтров к изображению (grayscale, blur) |
| `task5.html` | Коллекция различных элементов HTML форм |
| `task6.html` | Пример адаптивной верстки с header, main, sidebar и footer |

## Используемые технологии

- **HTML5** — семантическая разметка
- **CSS3** — Flexbox, Grid, 3D трансформации, анимации, медиазапросы, CSS фильтры
- **JavaScript** — работа с DOM, обработка событий, динамическое изменение стилей
